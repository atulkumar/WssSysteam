The screen shot below shows the Form Entry Screen. It has five Panels: Request Information, Project Details,

Technical Worksheet, Tab 4 and Tab 5.

 

Each Panel has text boxes, where we can add information that we wan to display. It has DateTime and MultiLine

 text boxes.

 

Various Panels in the Form Entry Screen:

1-      Request Information

2-      Project Details

3-      Technical Worksheet

4-      Tab 4

5-      Tab 5

 

 

 

 

 

1.         Request Information Panel

 

 

Form Name: Give the name of the Form. For Example, Leave Form, Human Resource

Fill the information in the fields and .then select the “Yes” Radio Button to display the field onto the Form.

If we select “NO”, the filed will not be displayed on the Form.

If a Date is to be displayed, select the DateTime Field. For Example, Date Of Birth.

 

Other Panel are also same. Tab 4 and Tab 5 are provided if incase, more information is to be displayed

in the Form.
