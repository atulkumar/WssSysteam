SubCategory:-

This module is used to create the SubCategory. Each SubCategory has some SubCategory members & an agreement

associated with it .These SubCategory members can assign tasks for call of SubCategory. The below screen shot shows

all the listed SubCategorys for various companies

 

  

 

 

A) HOW TO CREATE A NEW SUBCATEGORY

To create a new SubCategory click the Add New Button and it will open the SubCategory Detail Window. Here user can

enter the details and save the new SubCategory.

 

 

     

 

Customer

It has some pre-defined values i.e. the names of the companies like ION, LF.

 

Subcategory Name

User can give any name to the Subcategory.

 

Subcategory Type

It has some pre-defined values. User can select it from the drop down. It can be JDE, .Net, Web.

 

Parent Subcategory

Its optional and it again has some pre-defined values which can be selected from the drop Down.

 

Subcategory Owner

Its mandatory field. User can select the owner of the new Subcategory from the drop down.

 

Start Date

User can set the start date of the Subcategory from the calendar.

 

Close Date

User can also enter the Close Date of the Subcategory from the calendar. Its optional.

 

Status

It has two options either Enabled or disabled, which can be selected from the drop down.

 

Category code 1

Its optional Field.

Category code 2

Its optional field.

Description

User can add any description about the Subcategory

 

After entering all the details for the new SubCategory, User can save the SubCategory by clicking on the SAVE button,

After saving, user will get a message on the top of the window that records have been saved successfully.

 

After saving the new SubCategory, user can add the SubCategory members from the SubCategory Member panel. User can

            Add as many Members from the Drop down.

 

 

           

 

 

  Name

User can select the name of the member from the drop down menu. It has some Pre-defined values.

 

 

  Role

Each user has its own pre-defined Role, which can be selected from the drop down.

  Reports To

From the Drop Down menu, we can select the name of the person to whom the Subcategory

Member will report.

 

 

 

Agreement Lines panel details are optional. User can enter the details like Budget, Key Deliverance and description.

 

After entering all the details, click the SAVE button to save the records and the new SubCategory will be saved. The

Name of the new SubCategory will automatically appear in the entire drop down menus. In call entry screen, while saving

A new call, the name of the SubCategory will appear in the drop down menu.

 

NOTE: We can also create Views to view the details. For more details check Views.
