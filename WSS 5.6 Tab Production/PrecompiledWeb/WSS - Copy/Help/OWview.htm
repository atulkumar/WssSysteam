This screen lists down all the views and from here we can assign it to various companies and roles.  The View Detail

panel lists down details like View ID, which is a unique no, View name, Screen name i.e. on which screen it will be

reflected, Role name, comp name.

 

From New View Info, we can select and assign the view.

New View Name        Enter the name of the new view.

View Type                 It has some pre-defined options in the drop down menu. If we select Call View,

                                             then it will display all the views for call view.

Company Name         It opens up a window with all the companies name. It has some pre-defined options .

Role Name                 It opens a window and has some pre-defined options We can select roles like QA, IT, sales etc.

                     Select any view from the list of the views and then click on Copy View From to copy the view,

give a new name and then click on the Save button to save the new view.

 

             

 

To delete a View, click on the Delete button. To refresh the page, click on the Refresh button

 

                        Buttons

             SAVE                   After, entering the new data, click on the SAVE button to save the changes.

 REFRESH             It will refresh the page, all changes that were not saved will be deleted.

 DELETE                 It will delete the selected record
