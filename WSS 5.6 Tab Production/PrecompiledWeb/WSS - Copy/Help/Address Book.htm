Purpose:

The purpose of this screen is to give each person, customer, company or business location a unique

identity in the system.

 

 

 

 

 

 

 

            Buttons Used

                       

 

 CLOSE BUTTON        DELETE BUTTON                 SEARCH BUTTON     RESET BUTTON

 UPDATE/EDIT BUTTON      ADD BUTTON            SEARCH            

 

Disable Address Book        This button will be used when you have to disable any employees Address Book

 

 

Add No             Address No.

 

Name              Name of the New User or the Company.

 

Status              It has some pre-defined types. Disabled (DISA), Enabled (ENA), Hidden (HIDE),

 Visible (VSBL).

 

AddBook Type  It has some pre-defined types. Company (COM), Customer (CU), Employee

 (EMP), Manufacturer (MANF), Supplier (SPL), Vendor (VND).

 

Address1         Physical Address of the User or the Company.

 

To create a New Contact Details of a New User or a Company(Support Company/Client

Company), Click on  Add New Button. It will open up a New Form , where you can

enter the Name, Address, City, Province, Postal Code, Country, Email, Phone No etc.

   HELP                  Gives the brief description of the current screen.

  LOGOUT              Used to logout from WSS.
