This screen shows closed tasks by default.

 

Single clicking on any task shows the action filled.

 

 

 

 

 

 

 

 

Double clicking on any task opens the call related with that task.

 

 

 

ADD NEW         

It opens the Call Details window, we can add new call.

EDIT

Edits the changes in the selected record.

 

 

SEARCH

Type any keyword in the search boxes and then click on this button it will only display the matching result.

 

ATTACHMENT

Placing mouse on this button shows if there is any attachment attached or not. If attachment is there it will show View Attachment and if attachment is not there it will show no attachment uploaded.

 

CALL FORWARD  

We can forward the Task to the other person, using this button.     

 

 

VIEW OPEN/CLOSE CALLS  

 

Clicking on this button will show only open calls.

DELETE 

 

 

Deletes the selected record.

CLOSE 

 

 

It closes the current screen.

HELP

 

 

To view help Instruction on this screen.

LOGOUT

It is used to logout from WSS.
