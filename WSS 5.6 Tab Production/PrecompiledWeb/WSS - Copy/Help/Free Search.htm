Search is used to search and fetch a specific word or words from the entire database. We can also do an

advanced search by selecting the levels of the search.

 

 

Free Search is used for searching with a keyword, by selecting the various criteria’s

In the search criteria, we can select the company, project and the domain from which we want the

keyword to be searched.

 

The Company has some pre-defined values from the UDC. For example, ION, LF etc.

The Project also has some pre-defined values from the UDC. For Example, WSS, Anaha etc.

In the Domain, we can select the level from where we want to search the keyword like call, task, action,

Call and Task, Call and Action etc.. It has some pre-defined values from the UDC.

 

NOTE : Selecting the Company is mandatory. Project and Domain are optional.

 

Advanced Search

 

 

In the Advanced Criteria Panel, we can select some advanced conditions and criteria’s for the search.

We can select the Call no “From” and “To”. Here, we can put the call no’s from where we want to search

the keyword.

 

We can also select the Date “From” and “To” i.e search in all the calls for the keyword specified between

the date.

Levels

 

We can select the levels of the search i.e. at call level, task level or at action level.

If we select the call level search, then it will search for the specified keyword in the call subject. Description,

comment, attachment, Call type etc.

 

We can select the options in the check boxes.

For Example, if we want to select for the word which is in the subject, then we can check the subject check

box and it will search for the particular specified subject. Similarly for others.

 

The result of the Search is displayed in the results panel.

In the above screen shot, search was made on the word “Testing” for the company ION,  in the domain “call”.

Search was done at the call, task and action levels and the result was given showing two calls that contained

the word “testing”. We can click the result data to view the details of the call.

 

NOTE: For searching, if we are not sure of the complete word then we can add strings also.

For example, *te*. After search, in the results it will show all the calls which contain the alphabets “te”, no

matter what data they have in the front and back of “te”.

 

Another Example, if we want to search for a word starting from “Test” and ending at anything, then  we can enter

“Test*” in the search box. The special Character * will indicate that the word should start from Test and it can end

with anything, so it will look for all the word starting from test.

 

Another Example, for the word “Automation”, we can simply enter Auto*.

It will search for all the words in all the calls starting from “auto” and ending with anything.

 

Buttons

CLOSE                   It will close the current window.
