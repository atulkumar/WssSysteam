Comment

 

Clicking on comment, user can add any comment at task level.

 

 

Forms

Forms assigned to particular call type and task type can be filled by    clicking on the form button.

 

Attachment

Any attachment at task level can be attached clicking on attachment icon.

If this option is checked then an action has to be filled against that task showing the estimated time taken to complete the task.    

 

 

Status

It has certain predefined options such as Assigned, Reopen, Closed etc   to represent task status.

It has certain predefined options such as Coding, Admin (administration), Config (Configuration) etc.
