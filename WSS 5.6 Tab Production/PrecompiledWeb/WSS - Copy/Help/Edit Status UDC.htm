To Edit status UDC

 

To edit a Status details click on the Edit button and it will open up a Status UDC Edit window. After

making the required changes, click on the save button to save the changes.

 

The fields  screen name and company  are not editable, they are only read only. Other details

can be edited.

 

 

After editing the details click the Save button or Ok button.

 

 

   OK                      If clicked on OK button this will save your entry and take you to the call view screen

 SAVE                  After, entering the new call, we can click on the SAVE button to save the new call.

 REFRESH             It will refresh the page, all changes that were not saved will be deleted.

  CLOSE                  If we want to cancel the changes made and close the current window.
