Explanation

 

BUTTONS

 

 ADD NEW               It opens the Call Details window, we can add new call .

 REFRESH               It will refresh the page, all changes that were not saved will be deleted.

 EDIT                    It will open the call edit screen and we can edit the call and its details.

   DELETE                It will delete the records

 

 SHOW/ HIDE        If we click on this button once, it will SHOW call the calls and if we click it second time,

 it hides all the closed calls and only shows the calls in progress and assigned status.

 SEARCH                Type any keyword in the search boxes and then click on this button, it will only display

  the matching results.

 ATTACHMENT        With this button, we can attach files with the call or task.

 

 

 TASK FORWARD    We can forward the Task to the some other person, using this button.
