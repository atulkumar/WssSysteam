Document View

 

Document View

 

1. Click on Documents à Document View 

 

 

 

Add Button:-Click on Add button to upload new File, Document Detail will appears from where you can upload new file.

 

Edit Button: - By Press on Edit Button user can edit the existing file. Document Detail will appear in edit mode.

Delete Button: - By clicking the delete button user can delete the file along with all the file access permissions (if present).

 

Search Button: - It can be used to search the Grid contents.

 

Reset Button: - It will reset all the Contents entered for Searching.

 

 

 

1. Uploading New File

 

Click on ADD button from Document View screen to upload new file. Document Detail screen will open which will appear like following:-

 

 

 

 

Choose Group from Group Combo (Under which you want to upload the document).Enter Description and Click on Browse button to locate the file which you like to upload. Then click on Upload button to upload the file.

 

Note: - *A new Version will be saved if file is already exists under that selected group.

 

File Info will open which shows the information about uploaded file. You can edit the file description, you can Change the Group of file (it will move file from existing group to new group).

 

File Permissions Panel will open from where you can give access permissions to Role/User for the uploaded file.

 

 

*Note: - By selecting access level you can assign permissions to various users or you can assign the file access permissions to Role (All the users under that role will automatically get the permission to access that File).

 

 

 

The following screen will appear, Choose Company, status, Role Level from combos. You can choose Valid from and Valid to Date and click on Checkboxes to give access to User/Role.

 

 

 

 

Click on Save Button to save the File access permissions .The following screen will appear which will show the Screen Access Info panel.   

 

 

 

 

 

 

 

 

 

 

 

 

 

 

2. Editing the Existing File

 

Double click on Grid row on Document View screen to edit the existing file. Document detail screen will open in editable mode so that you can edit the file Info or you can assign new access permissions for selected file.  

 

 

3. Deleting  Existing File

 Click on Grid row to select the particular record. Now click on delete button to delete the selected record. All the file access permissions will be deleted (If any) along with File.

 

Group View

 

1. Click on Documents à Group View 

 

Group View screen will appear which will show all the groups.

 

 

 

Click on Folder to see all the uploaded files with file information under that folder (Group).Download link is available for file so that user can download file.
