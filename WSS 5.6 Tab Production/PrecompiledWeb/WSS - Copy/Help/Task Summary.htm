Task Summary:

 

This report is use to display the Hrs according to the company

 

 

           

 

 

Working of the report:

Depending upon ones requirement, select the criteria accordingly.

Company:- Select the name of the company

Subcategory:- Select the subcategory

Call Requested By:- Name of user who requested this call

Task Owner- Select the name of the user to whom the task was assigned

From Date- To:- Select the date range

Level1, Level2, Level 3:- Select what level you want to view this report. You can select only 1 out of 3

 

 

Once you have filled all the information you can hit the search button  This button will help you in displaying the
