To view the activated form for any company

 

Once the Forms have been created, we can simply select the Company Name from the drop down

menu and it will display a list of all the Active Forms for selected company. User can Double Click

 the Form and Preview the Forms, but cannot make any changes to the Form. we can assign the

Forms. For this, we have to select the Company Name

Then, click on the SEARCH button , to list down all the activated Forms.
