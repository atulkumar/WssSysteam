i)         Contact Info

 

 

 

We can add more Information in the Additional address Info, Category Info, Personal Info and Skill set

ii)        Additional Address Info

 

 

Additional information regarding the address can be added here. To add more addresses click on

iii)       Category Info

 

User can enter the various category codes. Its optional.

 

 

iV)       Company Info

 

This panel will be enabled only if the Address Book Type is “company” User can type in the details of the

company.

 

Company Type:           It has some predefined options. Click on the Plus sign and it will open the UDC

options in a new window.

 

Opened Date :             Enter .The date on which the company was opened. User can select the date from

the Calendar by clicking the calendar Icon.

 

Working Hrs Fr/To:     Enter the Working hours of the company, so that some other user/client can

check the timings if he wants to contact.

 

No of Employees:        Enter the total no. of Clients in the company.

 

Currency                     Enter the currency in which this company deals with

 

Rest of the fields are not mandatory

 

 

 

V)        Personal Info

 

 

 

 

Under Personal Information, user can enter the personal details like First, Middle, Last Name.

Date Of Birth:            It can be selected from the calendar by clicking the Calendar Icon.

 

Gender:                     It can be either Male/Female. These options are already predefined in the UDC.

Click the plus sign  and the options will open in  anew window.

 

Marital Status:            It has some predefined options in the UDC.

 

Working Hrs:              User can specify the Working hrs.

 

Joining Date:              User can select the joining date from the Calendar

 

Role:                         It has some predefined options in the UDC.

 

Time Zone:                User can specify the Time Zone from the drop down box.

 

Picture:                     User can add a picture also.

Mail Communication    If this box is checked then a mail will go if any changes are made in this Profile. This mail will

After entering all the necessary and required information in the Form, click on the SAVE button   to save

the new User/Company detail. Once the Record is saved, you will get a Confirmation Message at the top of the

Window.

 

Buttons Used

  OK                      Save the changes and Exit from the current screen.    

  SAVE                 After, entering the new data, click on the SAVE button to save the changes.

  CLOSE                 Close the current screen.

  RESET                 It  will Refresh the page and will discard all the unsaved changes.           

  EDIT                   Edit the changes in the selected record.            

  HELP                   Gives the brief description of the current screen.

  LOGOUT              Used to logout from WSS.
