This screen is a tool which will work as a communication between many people related to the

task/call/action. You will find comment icon  on call level, task level and action level.

 

Double click on the icon and it will display a comment screen with the call No. and Task.No

 

Write a comment in the given column and press OK or SAVE button.

 

 

 

Comment To- If you want any particular should be allowed to read the comment written

by you then you can do so by selecting the user from the drop down box

 

External/Internal- You will find two options under this drop down field where external means

if you want to make your comments public Whereas internal means to make your comments

private (only you can view your comment)

 

Send Mail- This button is used to send the comment through email to any concerned person

Double click on this button and a new pop-up screen will open named as WSS users.

Select the email ID to whom you think the mail should go and then click on OK button

 

You will come to do the comment screen now save the comment . Once you save the comment

icon which was earlier appearing in grey turned out yellow and with lines  once the comment

is read the line will disappear and the icon will look like this
