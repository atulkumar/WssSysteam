How to receive and clear the invoice

 

 

 

 

  If the user wants to cancel the invoice can do so by clicking the cancel invoice button placed on the first screen (search screen)

 

                          This button is used to cancel the invoice. Once canceled the invoice will go to Drafts screen

 

                          This button will show the clear and temporary invoices.

 

 

Double click on the invoice which will take you to the enter Invoice screen. 

 

 

 

 

This screen will ask user to fill the amount received against that invoice . Whatever amount the user fills under Amt Received

field will be calculated will reflect on other fields for example Exchange rates, taxes and Actual received fields. If there is any

balance it will be reflected under Balance field.

 

Now click on save button  only if you accept this invoice

 

 

  

 

  The information will be saved below after clicking on the save button. Once saved user can edit the information by double

  clicking on this grid.
