Call Status Summary:

The purpose of this report is to display the status summary report of a call in a graphical view

 

 

 

Working of the report:

Depending upon ones requirement, select the criteria accordingly.

Company:- Select the name of the company

Subcategory:- Select the subcategory

Call Requested By:- Name of user who requested this call

From Call No- To:- Filter the call according to the range

From Date- To:- Select the date range

Once you have filled all the information you can hit the search button  This button will help you in displaying the

Summary of call status in graphical view. Please see the screen shot below

 

 

You can further view more information on each call type by clicking on the blocks for. Exp click on the DNET bar and will give

you the detail graph of DNET call type.
