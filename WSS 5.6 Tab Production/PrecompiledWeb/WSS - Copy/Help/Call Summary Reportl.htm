Call Summary Report:

The purpose of this report is to display the summary of a call

 

 

Working of the report:

 

Depending upon ones requirement, select the criteria accordingly.

Company:- Select the name of the company

Subcategory:- Select the subcategory

Call Requested By:- Name of user who requested this call

From Call No- To:- Filter the call according to the range

From Date- To:- Select the date range

Once you have filled all the information you can hit the search button  This button will help you in displaying the criteria

you have selected. Please see the screen shot below

 

 

 

 

          

            Blue color on the call number indicates that there is further more information regarding that call. Hand image will appear to

click on the call number

 

Now hit the call number which will show you the detail of that call No.
