To enter new template, this screen will have 3 drop down panels

 

 

A)                Template Detail - Under this panel we will have following fields

 

 

 

NOTE:- There will always be three type of Template

 

1.       Call and Task Template (CNT) If we select CNT type of Template then unless and until

                  user don’t fill call detail information he cant move to the next level or panel which is

                  Task Detail. For this template you have to create a call and assign task.

2.       Call Only Template (CAO)- User is only need to fill the call detail information

3.       Task Only Template (TAO)- User can straight away go to third level or panel i.e Task Detail

      panel to

4.       Create the task template after filling the information in template detail.

 

Template Name- Enter the name of the template make sure you don’t enter the duplicate name.

Template Type- Select the template type

Customer/Company- For whom you want to create this template for

Call Type- Select the task type for which this template should be made

 

 

 

B)                Call Detail- This panel will be same what we have in our call detail screen

 

 

The following fields will be automatically filled as soon as we fill the template detail information

1.             Call Type

2.             Customer

 

 

Call Type:                   It has some predefined options in the UDC. Click on the Plus sign, and a pop up window will

open, we can select the call type from there.

 

Call Date:                        By default, it automatically picks up the current date.

 

Call Entered By:             By default, it takes the current user name.

 

Priority:                    It has some predefined options in the UDC like-Normal, Urgent, Minimum,

                                    Maximum, High and Low.

 

Customer:                  It has some predefined options in the UDC. 

 

Call Owner:                It has some predefined option from the UDC. On clicking the plus sign next to

                                               it, will open a pop up window and we can select the option from there.

 

Project:                     Select the project for whom you want to create this template F.ex. WSS

 

 

Template Type:             There are three types of template. They are predefined in the UDC. On clicking

the plus sign, a pop up window will open and we can select the template type

from there.

 

Template:                  It has some predefined options. Every template has some predefined tasks and

actions (depending upon the type of template), so we can select the appropriate

template depending upon the new call.

 

 

 

 

Cat Code 1, Cat Code2, Cat Code 3— It accepts only Numeric values.

 

Status:                      When we open a new call, then the status is OPEN, When we assign the call to the

task owner, then the status is ASSIGNED.

 

Subject:                    Enter the Subject of the call.

 

Description:               Give a brief description of the call.

 

 

Attachments:            We can attach files as attachments with this call. Click on the Attachments Icon

, a pop up window will open, we can Browse the file and then attach it.

 

 

 

c)                 Task Detail- Under this panel we will have following fields

 

              

 

 

 

 

C:                   Comment Once the task is entered, the icon for the comment appears. On clicking the Icon,

                      a window appears where we can enter the comment and save it.

 

A:                   Attachment If under this column, an Icon is displayed, then it indicates that this Task has

                      a file attached with it. Click on Icon , and it will open a window with a list of all the attachment

                      files for this Task. To View the files, Double click the file and it will open the file.

 

F:                   If under this column, an Icon  is displayed, then it indicates that this task has a form associated

                      with it. Click on the Form Icon  and it will open a window with a list of all the forms associated with

                      this task.

                     - This  Icon indicates that the form has no data.

                      - this Icon indicates that the form has data.

 

Subject          Enter the Subject of the call

 

Dep              If any dependency is defined to any task for example task has 1 dependency as (1) and task 2 has (2)

                    dependency in that case you can only fill in action under task 2 until and unless you don’t finish task 1 and

                    close it.

 

TType          Task Type-Select TType from the drop down menu.. It has some predefined options.

 

Owner          Select the Owner from the drop down menu. It displays a list all the users.

 

Dep              It’s optional. Select the value from the drop down menu.

 

 

EHr              Estimated Hours-enter the estimated hours needed to complete the task.

 

Act               If this check box is checked, then it becomes mandatory to enter the Actions for this task. Unless and

                    until, actions are filled for this task, task cannot be closed. Incase of mandatory, “M” is displayed,

                    indicating that Actions are to be filled before closing the task.

 

Pri               Priority-It indicates the priority of the task. It has some predefined options which can be selected from the drop

                   down menu.
