Purpose:

 

The purpose of the Template Entry Screen is to create a predefined list of call and task that will

help to populate the list of task under that particular call only if its type matches.

 

Template entry will come from Administration Center Menu. Below is the template view screen

 

 

 

 

This is a searchable screen and will help the user to do a fast search for any template created so far.

 

1.                  To create a new template click on  add button. This button will take you to template

detail screen where you can create your own template.

 

2.                 To delete any template click on  delete button. NOTE if the template is in use then it

cannot be deleted

3.                 To edit the exist template click on  edit button OR double click on that template it

will take you to the edit screen.
