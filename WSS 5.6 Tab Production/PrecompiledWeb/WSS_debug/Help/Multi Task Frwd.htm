Multi Task Forward

 

This screen will be used to forward more then one task to the other user.

 

 

 

 

Company

Select the name of the company from where you want to select the task.

Subcategory

Select the subcategory.

Task Owner

Select the owner of that particular task.

Assign To

Select the user to whom you want to forward those selected tasks.

 

 

Send Mail

If its checked means a mail will be sent to the user to whom the tasks have been forwarded BUT

If it’s unchecked then no mail will be sent.

 

 

                       

            

 

           

 

                 Please Note: - To forward the task select the check box and then click the save button

 

                Buttons used on multi task forward screen

 

 

 

 

SAVE

 

 

This button saves the record.

 

CLOSE 

 

This button closes the current screen.

 

 

HELP

 

To view help Instruction on this screen.

 

 

 

LOGOUT
