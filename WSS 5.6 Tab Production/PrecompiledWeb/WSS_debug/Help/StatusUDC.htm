It contains the details of the various Statuses which are used for call and Task. Here, we can add more

statuses according to need and requirement.

 

 

 

            To add a new UDC

To add a new status in the UDC, enter the name under the fast enter of Status UDC Detail, and then give the

description under the fast entry of Description. Then enter the status code. For example, ASSIGNED status

always comes in between OPEN and PROGRESS. So the status code can be any number between 10and 20.

 

After entering the details of the status, save it by using the Save button. The Statuses with green background

cannot be Edited, where as ones with white background can be edited.

 

Select screens-- It has some pre-defined options. We can select the options from the drop down menu. It lists down

the screen on which the status will be visible for calls and tasks.

 

Select Company – It also has some pre-defined options of the various companies. We can select the options from the

drop down menu. For Example, ION, LF etc.

 

Buttons Used

  SAVE                After, entering the new data, click on the SAVE button to save the changes.

  EDIT                   Edit the changes in the selected record.            

  RESET                 It will Refresh the page and will discard all the unsaved changes.           

  DELETE               Delete the selected record.

  HELP                   Gives the brief description of the current screen.

  LOGOUT              Used to logout from WSS
