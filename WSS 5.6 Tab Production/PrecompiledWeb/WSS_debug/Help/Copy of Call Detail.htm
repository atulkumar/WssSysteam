Purpose:

 

All things in WSS start with a CALL. A CALL can be an error, a question, a project task etc. A CALL is the

log of something needed to be executed.

 

 

 

Call Entry is done on this screen. After entering all the information, we can save the call, then we can fill

the corresponding tasks and actions for this call.

 

Call Type:                    It has some predefined options in the UDC. Click on the Plus sign,

 and a pop up window will open, we can select the call

type from there.

 

Call Date:                     By default, it automatically picks up the current date.

 

Call Entered By:          By default, it takes the current user name.

 

Priority:                       It has some predefined options in the UDC like-Normal, Urgent, Minimum,

maximum, high or Low.

 

Customer:                    It has some predefined options in the UDC. Click on the Plus sign and the pop

up window will open. We can select the option from there.

 

Call Owner:                  It has some predefined option from the UDC. On clicking the plus sign next to it,

will open a pop up window and we can select the option from there.

 

Est Finish Date:           Here, we can specify the finish date of the call. On clicking the calendar icon,

a calendar will open, and then we can select the date from there.

 

Project:                       We can enter the name of the project. F.ex. Ion

 

 

 Reference:                   To give any reference to your call for quick search

 

Agreement:                 It has some predefined options from the UDC. We can select the agreement

no/Type from the UDC.

 

Template Type:           There are three types of template. They are predefined in the UDC. On clicking

the plus sign, a pop up window will open and we can select the template type from

there.

 

Template:                    It has some predefined options. Every template has some predefined tasks and

actions (depending upon the type of template), so we can select the appropriate

template depending upon the new call.

 

 

Cat Code 1, Cat Code2, Cat Code 3— It accepts only Numeric values.

 

 

 

 

Status:                         When we open a new call, then the status is OPEN, When we assign the call to

the task owner, then the status is ASSIGNED. When, the task owner enters the

respective tasks and actions on the call, then status becomes PROGRESS. And

finally, when all the task are closed, then status is CLOSED. We can also re-open

the call, then status will become RE-OPEN.

Subject:                       Enter the Subject of the call.

 

Description:                 Give a brief description of the call.

Comment:                    We can add a special comment on the call. Click on the Comment Icon 

  and a pop up window will open. We can type the comment there

and save it. After adding the comment, the color of the Icon changes to Yellow,

indicating that there is a special comment attached here. If the comment is unread,

then in the icon , there will be three-four black lines, indicating that the comment

is still unread and after reading the comment the black lines disappear.

 

 

Attachments:               We can attach files as attachments with this call. Click on the Attachments Icon

, a pop up window will open, we can Browse the file and then attach it.

 

 

 Buttons

 

OK                                    If clicked on OK button this will save your entry and take you to the call view screen

 SAVE                                After, entering the new call, we can click on the SAVE button to save the new call.

 CANCEL                             If we want to cancel the changes made.

 REFRESH                           It will refresh the page, all changes that were not saved will be deleted.

 DELETE                               It will delete the records.

 ATTACHMENT                     With this button, we can attach files with the call.

 CALL FORWARD                 We can forward the call to the other person, using this button.                                                      

 

 

 

A)        How to Create a New Call

Enter the new call data, attach any files if needed or comments, then click on the SAVE button

 to save the new call in the records. The status of the new will be OPEN, once the task

 is assigned, then the status will automatically become ASSIGNED. Further, on entering the Actions,

 the status of the call will become PROGRESS.

 

 

 

 

 

 

Once the call is saved, we can fill the Tasks on the same screen in the Task Panel. And after filling the tasks,

we can also fill the Actions in the Action Panel on the same screen.
