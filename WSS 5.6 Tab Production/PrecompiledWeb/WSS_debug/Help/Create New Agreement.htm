A)        How to Create Agreement

 

 

To add new agreement click on Add button on clicking this button it will take you to the Agreement Detail screen

Let’s take an agreement which is already created .Will pick Nordic company’s agreement. Double click the grid line

and it will take you to the edit mode of agreement under Agreement detail screen. See the screen shot below

 

 

 

 

Customer:              It will ask you for which company you want to create an agreement. Select one out of the listed Company

                               I selected ION .

 

Project                   Select the project name for which you want to create this agreement for.

 

Contact Person      Who should be contacted regarding this agreement

 

Currency                 Currency will be picked automatically from address book if you have done the entry while filling

                               the company info. If you failed to do so, you can always mention it here BUT it will prompt you

                               the message to give the currency when you defining the skill levels.

 

Agreement Type   This field will tell what type if agreement its going to be Annual, monthly or yearly

 

 

Status                   The status here will tell you whether it’s going to be an active agreement or inactive. You create an

                              agreement but you want to get it approved before actually using it. For that you can select inactive.

 

Valid From/To      Validy date of this agreement for example from what date to what date this agreement will work

 

 

 

 

 

 

 

 

B)        How to define skill levels based on call/task type

          

 

AGREEMENT LINES

 

Under this panel you can define the skill level and the price for each level based on call/task

 

CallType                            Select the call type for which you want to do the pricing for it is a UDC field

 

TaskType                          Select task type , these are the UDC fields

 

Skill Level                         It’s again UDC field level can be Junior or Senior

 

Price                                  Mention the Price rate for the criteria      

 

Charge Basis                     There are two check boxes hourly or fixed, means you want price to be calculated based on hrs

or it should be fixed

 

 

Now that you have finished filling your agreement click on save button.
