Call Task Detail Report:

 

This report is use to display the details of all task related to a particular call no. If user wants to see the actions also,

then he/she has to click on the task, it will display the actions details of that selected task

 

           

 

 

Working of the report:

Depending upon ones requirement, select the criteria accordingly.

Company:- Select the name of the company

Subcategory: - Select the subcategory

Enter call No: Enter the call number you want to view in detail.

From Date- To:- Select the date range

 

Once you have filled all the information you can hit the search button  This button will help you in displaying the

Call Task Detail Report. Please see the screen shot below

 

Please Note:- The call number showing in blue color indicates that there is further information available for this call

 

 

 

Buttons Used
