Under this screen, a list of all the calls is displayed with their call no, request date, call status,

priority, subject etc. We can also fill the tasks for the respective calls. To enter tasks select a

call and single click on it, then the Task view panel will open. We can enter the tasks and we

can save it. We can also attach comments and attachments to a task.

                Under call View :

1)

We can Edit the details of the call. For this, Select the call, and then press the Edit button or simply double click the call. It will open the call edit screen and there we can make the respective changes. After all the changes have been made, click on save button, to save  all the changes.

 

2)

We can Edit tasks also. For this, select the tasks, and then press the Edit button or simply
double click the task. It will open the call edit screen and there we can make the respective changes. After all the changes have been made, click on save button, to save all the changes.

 

C:Comment

We can add a special comment on the call. Click on the Comment Icon  and a pop up window will open. We can type the comment there and save it. After adding the comment, the color of the Icon changes to Yellow with dotted lines  ,indicating that there is a special comment attached here. If the comment is unread, then in the icon , there will be three-four black lines, indicating that the comment is still unread and after reading the comment the black lines disappears and it appears only in yellow color . User will be able to add a comment only after saving the new call.

 

 

A: Attachment

To Attach an attachment user has to double click on the task, by doing so a  pop-up window for task edit Screen will appears. Under this screen an icon  is displayed, Clicking on this icon will further open another screen where user can click on the browse button and selects the file to upload and then save the attachment.
NOTE- You have to save the task again when attachment is uploaded..

 Owner   

 Select the Owner from the drop down menu. It displays a list all the users.

 

 

 

            Dep               

Dependency -It’s optional. Select the value from the drop down menu. If a task ha          Dependency value 1, it means this task is dependent on this task no 1 and until we fill actions for task no 1 and close it, we cannot fill actions for the present task.

 

 

 

 

EstClDate

Estimated Close Date-The close date of the calendar can be selected calendar. Click on the calendar Icon to open the calendar.

 

 

 

EHr  

Estimated Hours-Enter the estimated hours needed to complete the task.

 

AcM  

Action Mandatory-If this check box is checked, then it becomes mandatory to enter the Actions for this task. Unless and until, actions are filled for this task, task cannot be closed.   On saving the information ‘M’ (Mandatory) is displayed. If the box is not checked then it is not necessary to fill actions for the task and task can be closed without filling action.  On saving the information in such case ‘O’ (Optional) is displayed.

 

 

Prio 

Priority- It indicates the priority of the task. It has some predefined options which can be selected from the drop down menu.
