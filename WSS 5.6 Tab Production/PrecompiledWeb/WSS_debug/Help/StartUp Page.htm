After log in, this is the first screen that a user sees. It has a summary of the recently opened calls,

and the jobs in the To Do List along with the calendar.

 

           

 

 

A)        Recently Added Calls

 

It shows all the calls that were opened recently along with other details like Call No, Comp ID, Call Type,

Priority, Status, Description etc. From here, we can check the calls that are in OPEN and PROGRESS status.

On Double Clicking a particular call, it opens the Call Detail Screen, where we can see the details of the call

and also Edit the details.

 

B)        To Do List

It lists the jobs that the person is supposed to do along with details like call no, task no, Company,

Description, owner etc. From here, we can see the tasks that we have to complete.

On Double Clicking a particular Taskl, it opens the Call Detail Screen, where we can see the details

of the call, task and also Edit the details.

 

C)        Calendar

 

Current Date is highlighted. We can also add some reminder or event on any date. For this just click on the

date and it will open a window and we can enter the details of the event.

 

           

 

To Add an Event

For this, click the Add New button . It will open a new window, where we can add up details.

  CLOSE                 Close the current screen.

 

 

Event Type-   It has some pre-defined option. Select the option from the drop down menu.

 

Subject-          Enter the Subject of the Event.

 

Message-        Enter the message that you want to display when the event occurs.

 

           

After entering the necessary details, click on the Save button to save the event.

 

Buttons Used

  SAVE                    After, entering the new data, click on the SAVE button to save the changes.

  OK                      Save the changes and Exit from the current screen.            

  RESET                 It will refresh the page and will discard all the unsaved changes.  

  CLOSE                 Close the current screen.
