Document View

 

This screen is used to search and view the document uploaded under different company.

 

 

           

 

 

                        Buttons

 

ADD NEW             

It opens the Document detail screen

 

EDIT

It will allows you to edit the selection you have selected from the grid

RESET 

It will Reset the page, all changes that were not saved will be deleted.

DELETE

It will delete the selected record                                                                                           

 SEARCH              

Type any keyword in the search boxes and then click on this button; it will only display the matching results.

HELP

To view help Instruction on this screen

LOGOUT

Used to logout from WSS.
