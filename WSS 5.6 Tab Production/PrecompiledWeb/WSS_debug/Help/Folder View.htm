Folder View

 

 

 

Note-If the user with which we have logged in belongs to Client Company then he/she will be able to view

Files and folders of that particular company.

 

If the user with which we have logged in belongs to Support Company then he/she will be able to view

Files and folders of all the companies on which he/she has access which is defined for his/her role

By default only add folder  button ,search and close button are enabled and any user has right to add a folder in a company.

 

Selecting a particular company and clicking on add folder button we can create folder under that company.

Further we can create n number of sub folders under a folder. Creator of a folder has all the rights on it by default, even if the role which is assigned to them has that right or not.

 

Once the folder is created, all other buttons gets enabled automatically for the creator of the folder. For others it depends upon the person assigned to his/her role/company

 

Above the various files of a folder can be seen the name of the company and the folder to which the file belongs in file list panel.

 

 Clicking on edit folder button we can make changes only in the description. Folder name can only be edited if it’s empty i.e. it does not contain any file/folder in it.

 

 Clicking on set permission button will further open a screen through which permission can be set on a folder only. Permission on a folder can be given to a company or a role. While granting different permissions we can check or uncheck allow/deny check buttons according to the need of the company or that particular role. But Set folder permission is denied as giving set folder permission means giving admin rights.

 

 Typing a particular text in text search box along with asterisk sign* and clicking on search button will show that particular text. This way the user can search for particular files easily. Same action can also be performed by typing the text and pressing enter button.

 

 Clicking on add file button will open a screen which has upload document provision, thus clicking on browse button we can add a file in a folder.

 

 

 Clicking on edit file button we can make changes only in the description.

 

 Clicking on close button will close the current screen.

 

 Clicking on help button will open the help for this particular screen.

 

 Logout button is used to logout from WSS.
