Purpose:

The Purpose of this screen is to give and take away the access of different modules, screens,

fields and buttons to different user/role of different companies. 

 

 

 

This screen displays the list of all the Role Names with their respective unique ID’s.

 

To create a New Role and Click on Add New Button. It will open up a New screen named as

“Role Permission” here you can provide the permission to that particular role.

 

Buttons Used

 

  ADD NEW            Create a new Role and it will open the new role window. 

  RESET                 It  will Refresh the page and will discard all the unsaved changes.           

  EDIT                   Edit the changes in the selected record.            

   DELETE              Delete the selected record.

   SEARCH             Used to search for specific results.          

   HELP                  Gives the brief description of the current screen.

  LOGOUT              Used to logout from WSS.
