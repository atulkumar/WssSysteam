1.       Select any UDC and click on update button  it will open a new window where you can

edit the already filled information. See the below screens shot

 

 

 

 

Product Code: Give any number

UDC Type: The type of UDC it is

Name: Abbreviation given for UDC

 

The above fields you cannot do the changes

 

Company: Name of the company that UDC belongs to

Description: Detail description of UDC

 

 

            After doing the modification either click on SAVE button or OK

 

            Buttons Used

  SAVE                 After, entering the new data, click on the SAVE button to save the changes.

  RESET                 It  will Refresh the page and will discard all the unsaved changes.           

  OK                      Save the changes and Exit from the current screen.          

  CLOSE                 Close the current screen.
