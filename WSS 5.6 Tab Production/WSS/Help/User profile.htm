Buttons Used on user profile screen

 

 

 

ADD NEW

 It opens the new window called user management window where details about new user can be filled.

 

RESET 

It will Reset the page and will discard all the unsaved changes.      

 

EDIT   

 

Edit the changes in the selected record.            

 

 

SEARCH

 

It is used to search for specific results.  

SHOW ALL / HIDE DISABLED BUTTON        

This button is used to show/hide the disabled records

 

DELETE  

 

It deletes the records not in use.

 

HELP

 

Gives the brief description of the current screen.

 

LOGOUT          

It is used to logout from WSS.
