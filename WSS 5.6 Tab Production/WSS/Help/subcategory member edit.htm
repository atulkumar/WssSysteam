SubCategory Edit-

 

            The purpose of this screen is to edit the information of Subcategory member.

 

 

 

Member Name

Names of the members associated with this Subcategory.

Role

Select the role.

Reports To

Select the name of the user to whom the member/employee will report to.

            

 

 

             Buttons used on subcategory member edit screen

 

 

OK                            

If clicked on OK button this will save your entry and take you to the Subcategory Detail screen.

SAVE 

This will save the information.

CLOSE

This button will be used to close the screen without making any changes.

HELP

This button will open a help file for this screen.
