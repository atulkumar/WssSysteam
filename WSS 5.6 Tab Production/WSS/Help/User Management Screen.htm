Enter the User ID for the New User, then enter the Password and Confirm the password again.

Address No  Once entry of the New User has been made in the Address Book, then it auto creates

an Address No. So, it has a pre-defined list, and

         

 

User Type                    It has two pre-defined options, Internal or External. We can select the appropriate option from

                     the Drop Down Menu

 

 

From Date          It is the Date on which the User profile is created. By Default it takes the

  Current Date automatically.

To Date             It is the Date, till when the User Profile will remain Active. We can select

  the Date from the Calendar.

                        We can click on the Calendar Icon to open the Calendar.

 

 

 

Now , we can select the Expiry Date of the User Account.

 

Status                Status has two pre-defined options—Enabled and Disabled.

 

 

 We can select the appropriate option from the drop down menu.

If ENABLED is selected, then the user will be able to Login. and

if DISABLED is selected, then the user will not be able to Login.

 

Password Expiry Limit --- It is automatically 356 days. We cannot change this field.

                       

                                               

            Admin Rights-      If this check box is checked in it means the user will have the rights to view the reports of all users.

                                     BUT if its unchecked it means he can only view his report.

 

              Mail Communication:-  If this is checked in it means the mail will go to the user whenever his/her profile is close to an

                                          expiry date . If its unchecked then mail wont work and no intimation will be send to the user

 

Roles                Select the type of role that you want to assign to the new user.

            For example, Sales Role, HR Role etc. 

                       

 

 

Role Name-      It has some pre-defined options . To see the options window, click on the small

plus sign.

 

Assigned Date  It automatically sets the current date.

       

 

 

 

Valid Upto        From the calendar, we can select the valid upto date. To open the calendar, click

on the calendar Icon.            

                                     

Status               Status has two pre-defined options—enabled and Disabled.

 

 

 

We can select the appropriate option from the drop down menu.

If ENABLED is selected, then the particular Role will get enabled and the user will

be able to Login with that Role. And if, DISABLED is selected, then the User will not

be able to Login with that Role. This is used when a User has been provided lot of Roles.

                         

Assigned By    By default, it picks the name of the user who is creating the New role.

 

Buttons Used

 

  SAVE                 After, entering the new data, click on the SAVE button to save the changes.

  OK                      Save the changes and Exit from the current screen.            

  RESET                 It  will Refresh the page and will discard all the unsaved changes.          

  CLOSE                 Close the current screen.

   HELP                  Gives the brief description of the current screen.

  LOGOUT              Used to logout from WSS.
