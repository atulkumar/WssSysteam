Call Fast Entry

 

This screen will be used by the client to save their time while logging any issue

 

 

 

 

 

Comment:- Can add comment here

Attachment- The user can attach a document

 

Subject- This field is used to give a brief description about this call

 

Description- Give the detail description

Call Type- Enter call type

Priority- Choose the priority level for this call

 

The client needs to enter just two information Call/Issue Subject and Call/Issue description.

 

Once these information is saved a mail will be generated automatically will be send to ION-CM

(Client Manager of support company) who further accept this issue and assign a call No to this issue

and again a new mail will be generated this time the mail will go the Client who actually to logged this issue.
