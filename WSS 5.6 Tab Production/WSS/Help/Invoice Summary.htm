Invoice Summary:

The purpose of this report is to display the invoice summary.

 

 

 

 

 Working of the report:

Depending upon ones requirement, select the criteria accordingly.

Company:- Select the name of the company

Select Status:- Select the status of the invoice you would like to view. You can select all

Once you have filled all the information you can hit the search button  This button will help you in displaying the

Invoice summary depending upon the status. Please see the screen shot below

 

Click on Show Invoice you will see the invoice

 

           

To see more in detail of this invoice, click the Detail image on this screen

 

           

           

When you click on MORE it will further give you the detail of each call.

 

 

Buttons Used

 

  SEARCH                          This button helps to search the information

 

  CLOSE                             Close the current screen.
