Under this screen, a list of all the calls is displayed with their call no, Request Date, Call Status,

Priority, description details. We can also fill the tasks for the respective calls. To enter tasks at

The Task View Panel, highlight the call, then the Task view panel will be enabled, and we can

enter the tasks there and then we can save it. We can also attach comments to the task and

can also attach files.

 Buttons on the CALL VIEW SCREEN

 ADD NEW             It opens the Call Details window, we can add new call .

 SAVE                   After, entering the new data, click on the SAVE button to save the changes.

 EDIT                  It will open the call edit screen and we can edit the call and its details.

 SHOW/ HIDE      If we click on this button once, it will SHOW call the calls and if we click it

second time, it hides all the closed calls and only shows the calls in progress

and assigned status.

 SEARCH               Type any keyword in the search boxes and then click on this button, it will

                     only display the matching results.

 ATTACHMENT       With this button, we can attach files with the call or task.

 REFRESH             It will refresh the page, all changes that were not saved will be deleted.

 Call Monitor         This button is used to put any call under monitoring

 CANCEL               If we want to go out of this screen use the cancel button.

 View My Call       This button is to view your calls only

 

 HELP                      Gives the brief description of the current screen.

             LOGOUT                 Used to logout from WSS.

 

Under call View :

 

1-               We can Edit the details of the call. For this, Select the call, and then click on the Edit button

          or simply double click the call. It will open the call edit screen and there we can make the

          respective changes. After all the changes have been made, click on save button, to save

          all the changes.

2-               Enter the tasks for the calls. For this, select the call, and then in the Task View Panel, the

          details of the Task will be listed. We can add more tasks to the call. Click save button to

          save the changes.

 

 

C:                   Comment-- Once the task is entered, the icon for the comment appears. On

clicking the Icon, a window appears where we can enter the comment and

save it. Comment can be Internal/External. Internal comment cannot be viewed by

the client but external comment can be viewed by all. User can also send email to

the specific users on comments. For more details check the Help on comment window.

 

A:                  Attachment-To Attach an attachment user has to double click on the task, by doing

so a pop-up window for task edit Screen will appears. Under this screen an icon  is

Displayed, Click on this icon which will further open another screen where user can

Click on the browse button and select the file he wants to upload and then save the

Attachment.

NOTE- You have to save the task again when attachment is uploaded..

 

F:                  Form-If under this column, an Icon  is displayed, then it indicates that this

Task has a form associated with it. Click on the Form Icon  and it will open

a window with a list of all the forms associated with this task.

- This  Icon indicates that the form has no data.

 - this Icon indicates that the form has data.

 

 

Subject           Enter the Subject of the call

 

TType             Task Type-Select TType from the drop down menu.. It has some predefined options.

 

Owner            Select the Owner from the drop down menu. It displays a list all the users.

 

Dep                Dependency -Its optional. Select the value from the drop down menu.

 

EstClDate       Estimated Close Date-The close date of the calendar can be selected from the

                     calendar. Click on the calendar Icon to open the calendar.

 

EHr                Estimated Hours-enter the estimated hours needed to complete the task.

 

Act                If this check box is checked, then it becomes mandatory to enter the Actions for

                     this task. Unless and until, actions are filled for this task, task cannot be closed.

                     Incase of mandatory, M is displayed, indicating that Actions are to be filled

                     before closing the task.

 

            Pri                 Priority- It indicates the priority of the task. It has some predefined options which

                               can be selected from the drop down menu.
