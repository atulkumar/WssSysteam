This screen displays all the Tasks along with their Status, Comments, Open date, Close date, attachments

and forms. If we select a Task, then it also displays the Actions for that task in the Action View Panel below.

We can see the Actions already filled and can also fill more Actions.

This screen displays the Tasks that a user has to complete along with the Target Estimated close dates.

User can directly enter the Action details and save the Actions.

We can click on Attachment Icon, to open and view the attachment files associated with a particular

call and task. Further, as discussed before, this Icon, indicates that a Form is associated with the

 Action Level

 

Description-    You need to enter the description in fast entry regarding to that corresponding task

Action Type-    This field is not mandatory. You can select the action type from this drop down box.

Act-                 This means whether actions Hrs are mandatory to fill if checked then it means its mandatory BUT if its

unchecked then it means you can omit hrs field.

Type-               There can be two type external/Internal If checked then it means the actions are external and can be

viewed by other users. If you uncheck this box then it means no one other than you can view the action

against this task.

 

BUTTONS

 

 SAVE                    After, entering the new call, we can click on the SAVE button to save the new call.

 

 ADD NEW              It opens the Call Details window, we can add new call.

 

 REFRESH               It will refresh the page, all changes that were not saved will be deleted.

 

 EDIT                    It will open the call edit screen and we can edit the call and its details.

 

   DELETE                It will delete the records

 

 SHOW/ HIDE         If we click on this button once, it will SHOW call the calls and if we click it second time,

    it hides all the closed calls and only shows the calls in progress and assigned status.

 SEARCH                 Type any keyword in the search boxes and then click on this button, it will only display

    the matching results.

 ATTACHMENT          With this button, we can attach files with the call or task.

 

 TASK FORWARD    We can forward the Task to the some other person, using this button.             

 

            CLOSE TASK            This button is used to close a selected task.

           

               HELP                      Gives the brief description of the current screen.

 

  LOGOUT                  Used to logout from WSS.
