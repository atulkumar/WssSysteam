This screen will be used to view the logs of any call/task/action

 

 

Company- Select the company Name

Call No. Type the call No.

Task No. Enter the Task No if you want to view the log of any task along with call

Call Level, Take Level and Action Level- You can choose either all OR the one you want to view the log for

 Search   To view the logs after all selection has been made click on the search button

 

 Reset    This button will be used to reset the screen   

 

Close     This button will be used to close the current screen
