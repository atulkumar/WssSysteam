Communication setup is designed to set up the mode to communicate with a person on specific events. 

The two modes used are either by Email or by SMS. We can specify rules , like on which events mail/sms

should be sent to the respective persons.

 

For Example, we can create a rule, if a new call is opened, then the call owner should get a mail. So, as

soon as a new call will be opened, call owner will receive a mail for conformation.

 

We can also create a rule, if any task is assigned or an Action is filled, Call owner should get a mail. So,

whenever a task will be assigned for that call, a mail will be sent immediately to the call owner. Similarly,

for Action filled.

A)        How to Create a Rule

 

 

RNo is the Role no, which is automatically assigned after the Rule is created.

M-Mail, click this check box for communication through mail

S-SMS, Click this check box for communication through SMS.

Start Date- it picks up the current date automatically

Stop Date – Its optional.

Event – From the drop down menu, we can select the pre-defined events.

For Example, Call Open, Task Assign, Action Filled, Task Fwd, Conditional, Additive, Call Closed, Task

Closed.

 

Additive Rule : This rule is used a new User to the Default rule.

 

Additive rule is created, if a call priority is HIGH, then a mail should be sent to “Nisha”

 

Conditional : This rule is used to put condition for communication, on the user for a default rule.

Example :

 

 

Conditional rule is created that whenever an action if filled, and if Admin is involved in the call cycle, then

no mail no sms should be sent to admin.

 

Default User-  from the drop down menu, we can select the user.

OnEvent- from the drop down menu, we can select the event. Its optional.

 

So, similarly, we can select the options and then Save the Rule.

C)        How to Disable a Created Rule

To disable a particular rule, select “Dsb” from the last drop down box. “Enb” is for Enabled.

On Selecting “Dsb” , the value in the last Column for that rule will become “0”, which indicates

that this rule is disabled and it will grey out also. If a Rule is Enabled, then the status column

will show “1” value.

 

             

 

D)        What are Alerts and How to set an alert

 

We can set Alerts on a particular call or task. With this if there is any change in the Status of the

Call or Task, a mail will be sent immediately. To set the Alert , select the Task then click on the

Alert Button, . Further, if any changes are made to the task or call, it will be notified

immediately.
