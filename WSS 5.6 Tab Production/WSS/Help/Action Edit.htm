Action Edit: - This Pop up screen is used to edit the actions filled.

                       

           

 

 

 

 

Action Date

By Default it takes the present date and time, but one can edit it as desired.

 

Action owner

Name of the person who is filling the actions.

 

Used hrs

Enter the number of hours taken in completing the action.

Description

One can write the description of the action etc.

Internal /External check box

It is used for invoicing .For billable action it should be external and checked.

Internal- If internal is selected then the action filled will be visible only to the user who has filled the actions.

External- If external is selected then the action filled will be visible to others

 

Used hr. mandatory check box

If this box is checked it is mandatory to enter the used hrs. If someone

does not want to mention the used hrs simply uncheck this box.

 

 

Comment

We can add a special comment on the task. Click on the Comment Icon 

      and a pop up window will open. We can type the comment there

                       and save it. After adding the comment, the color of the Icon changes to Yellow with dotted lines  ,indicating that there is a special comment attached here. If the comment is unread,  then in the icon , there will be three-four black lines, indicating that the comment  is still unread and after reading the comment the black lines disappears and it appears only  in yellow color . User will be able to add a comment only after saving the new task.

 

Attachment

 We can attach files as attachments with this task. Click on the Attachments Icon

                              , a pop up window will open, we can Browse the file and then attach it by clicking the Upload button. One can attach a file up to 7 MB along with the action.

 

 

           

 

          Buttons used on action edit screen

          

         

 

SAVE

 

It is used to save information.

 

OK

If clicked on OK button this will save your entry and take you to the To do list screen

 

RESET

It will reset the page, all changes that were not saved will be deleted.

 

CLOSE 

 

 

This button is used to close the current screen.

 

 

 

HELP

 

To view help Instruction on this screen.
