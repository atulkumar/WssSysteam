Action Details:

 

This report is use to display the details for the actions filled.

 

           

 

 

Working of the report:

Depending upon ones requirement, select the criteria accordingly.

Company:- Select the name of the company

From Date : - Select the date from which you want to see actions

To Date :- Select the date till which actions are required

 

Once you have filled all the information you can hit the search button  This button will help you in displaying the

Action Details Report. Please see the screen shot below
