Through this screen we can give access to various call types. If the check box is checked at call entry and call fast entry level it means we have access to those call type at both the levels.

We can check or uncheck the boxes according to our own choice.

SAVE

 

 

It is used to save information.

 

 

 

 

CLOSE

 

 

 

 

Closes the current screen.

 

 

 

HELP

 

 

To view help instruction on this screen.

 

 

 

 

 

LOGOUT
