A) HOW TO CREATE A NEW PROJECT

To create a new project click the Add New Button and it will open the Project Detail Window. Here user can

enter the details and save the new project.

 

 

Customer:                       It has some pre-defined values i.e the names of the companies like ION, LF.

Project Name:                 User can give any name to the Project.

Project Type:                  It has some pre-defined values. User can select it from the drop down. It can be JDE,

     .Net, Web for example.

Parent Project:               Its optional and it again has some pre-defined values which can be selected from the drop

     down.

Project Owner:               Its mandatory field. User can select the owner of the new project from the drop down.

Start Date:                     User can set the start date of the Project from the calendar.

Close Date:                     User can also enter the Close Date of the Project from the calendar. Its optional.

Status:                           It has two options either Enabled or disabled, which can be selected from the drop down.

Category code 1:             Its optional Field.

Category code 2:                           Its optional field.

Comment:                    User can add any comment for the new project. Its optional.

 

After entering all the details for the new project, User can save the project by clicking on the SAVE button, .

After saving, user will get a message on the top of the window, that records have been saved successfully.

 

 After saving the new project, user can add the Project members from the Project Member panel. User can

 add as many Members from the Drop down.

 

 

Name:                         User can select the name of the member from the drop down menu. It has some

  pre-defined values.

Role:                           Each user has its own pre-defined Role, which can be selected from the drop down.

Reports To:                  From the Drop Down menu, we can select the name of the person to whom the project

  member will report.

 

Agreement Lines panel details are optional. User can enter the details like Budget, Key Deliverance and description.

 

After entering all the details, click the SAVE button to save the records and the new project will be saved. The

name of the new project will automatically appear in all the drop down menus. In call entry screen, while saving

a new call, the name of the project will appear in the drop Down menu.

 

NOTE: We can also create Views to view the details. For more details check Views.

 

 

 

Buttons Used

 

  ADD NEW            Create a new project and it will open the project detail window. 

  EDIT                   Edit the changes in the selected record.            

  RESET                 It  will Refresh the page and will discard all the unsaved changes.           

   SEARCH             Used to search for specific results.          

  OK                      Save the changes and Exit from the current screen.            

   DELETE              Delete the selected record.

  CLOSE                 Close the current screen.

  SAVE                 After, entering the new data, click on the SAVE button to save the changes.

   HELP                  Gives the brief description of the current screen.

  LOGOUT              Used to logout from WSS.
